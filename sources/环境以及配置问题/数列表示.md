在 Lean 4 中，**数列**（即以自然数为定义域的函数）通常表示为从 `ℕ` 到某个类型 `α` 的函数：

```lean
a : ℕ → α
```

其中：

- `ℕ` 是自然数类型；
    
- `α` 是目标类型，例如 `ℕ`（自然数）、`ℤ`（整数）、`ℝ`（实数）等；
    
- `a` 是数列的名字，`a n` 表示第 $n$ 项。
    

---

### 📌 1. 数列的定义示例

#### 自定义一个数列：$a_n = 2n + 1$

```lean
def a (n : ℕ) : ℕ := 2 * n + 1
```

使用：

```lean
#eval a 0  -- 输出 1
#eval a 3  -- 输出 7
```

---

### 📌 2. 数列作为函数使用

你可以把数列视为函数，并像调用函数一样使用：

```lean
def b : ℕ → ℤ := fun n => (-1)^n * (n + 1)
#eval b 0  -- 输出 1
#eval b 3  -- 输出 -4
```

---

### 📌 3. 数列通项可使用 `match` 分段定义

```lean
def c : ℕ → ℤ
| 0     => 1
| (n+1) => 2 * c n
```

表示 $c_0 = 1$, $c_{n+1} = 2c_n$，即 $c_n = 2^n$

---

### 📌 4. 数列作为有限数组 (`Fin n → α`)：有限长度数列

如果你只想表示一个有限长的数列（例如前 5 项）：

```lean
def d : Fin 5 → ℕ
| ⟨0, _⟩ => 1
| ⟨1, _⟩ => 2
| ⟨2, _⟩ => 4
| ⟨3, _⟩ => 8
| ⟨4, _⟩ => 16
```

也可用 `List` 或 `Vector` 来表示：

```lean
def d_list : List ℕ := [1, 2, 4, 8, 16]
```

---

### 📌 5. 常用的 Lean 数学库定义方式（来自 mathlib）

在 `mathlib4` 中，经常用 `a : ℕ → ℝ` 来表示一个实数数列，例如：

```lean
variable (a : ℕ → ℝ)

example : ℕ → ℝ := fun n => a n + 1
```

---

### ✅ 总结

|表示形式|用途描述|
|---|---|
|`ℕ → α`|无限数列，常见于数学定义|
|`Fin n → α`|长度为 `n` 的有限数列|
|`List α`|可变长度的有限数列|
|`Vector α n`|长度固定为 `n` 的有限数列|
